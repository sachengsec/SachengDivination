# Sacheng Divination

专业六爻排盘在线工具，基于周易纳甲法，支持多种起卦方式，提供完整的卦象分析。

## 功能特性

### 起卦方式
- **自动起卦**：模拟三枚铜钱投掷，自动生成卦象
- **手动摇卦**：手动记录每次投掷结果（字/背），逐爻生成
- **时间起卦**：根据当前年月日时起卦
- **指定卦象**：直接从64卦中选择，可指定动爻

### 排盘内容
- **本卦与变卦**：自动计算并显示变卦
- **纳甲**：天干地支配爻
- **六亲**：兄弟、父母、官鬼、妻财、子孙
- **六神**：青龙、朱雀、勾陈、螣蛇、白虎、玄武
- **世应**：自动标注世爻、应爻位置
- **伏神**：显示伏藏之神
- **月建日辰**：自动计算干支历信息

### 分析功能
- **卦象概要**：卦名、宫位、动爻信息
- **世应分析**：世应五行生克关系判断
- **用神分析**：根据求测事项确定用神，分析旺衰
- **动爻分析**：动爻变化及回头生克
- **综合判断**：结合各项因素给出综合结论

### 其他功能
- **事项类别**：支持求财、事业、婚姻、健康、出行、官司、寻物、学业等
- **复制卦象**：一键复制卦象信息及AI分析提示词
- **响应式设计**：支持桌面和移动设备

## 使用方法

1. 打开 `index.html` 文件
2. 输入求测事项（可选），选择事项类别（可选）
3. 选择起卦方式：
   - 点击「自动起卦」立即生成
   - 点击「手动摇卦」逐爻记录
   - 点击「时间起卦」按当前时间起卦
   - 点击「指定卦象」选择特定卦象
4. 查看排盘结果和分析

## 技术说明

- 纯前端实现，无需服务器
- 基于传统纳甲法算法
- 64卦完整数据
- 八宫卦序推算
